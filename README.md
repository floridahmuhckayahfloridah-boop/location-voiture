# location-voiture

## English Overview

This project is a static website for sports car rental services offered by Magintec Sarl, located in Weiswampach, Luxembourg. The website features a contact page with an embedded Google Maps showing the company's location, a contact form for inquiries, company contact information including address, phone, and email, and a hero section with reservation instructions.

The site is built using HTML, CSS (with Bulma framework), and includes custom styling for a dark theme. It is designed to be responsive and user-friendly for potential customers looking to rent sports cars.

To view the website, simply open `index.html` in a web browser.

## Aperçu Français

Ce projet est un site web statique pour les services de location de voitures de sport offert par Magintec Sarl, situé à Weiswampach, Luxembourg. Le site web comprend une page de contact avec une carte Google Maps intégrée montrant l'emplacement de l'entreprise, un formulaire de contact pour les demandes, les informations de contact de l'entreprise incluant l'adresse, le téléphone et l'email, et une section hero avec les instructions de réservation.

Le site est construit en utilisant HTML, CSS (avec le framework Bulma), et inclut un style personnalisé pour un thème sombre. Il est conçu pour être responsive et convivial pour les clients potentiels cherchant à louer des voitures de sport.

Pour voir le site web, ouvrez simplement `index.html` dans un navigateur web.
